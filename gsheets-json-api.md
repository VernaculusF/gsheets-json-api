Разработай Python-проект для создания REST API, которое отдаёт данные из Google Sheets в JSON.


## ТЕХНОЛОГИИ:
- Python 3.10+
- Flask (или FastAPI - обоснуй выбор в комментариях)
- gspread
- google-auth
- python-dotenv

## ФУНКЦИОНАЛ:
- Авторизация через сервисный аккаунт Google (creds.json)
- Чтение таблицы по ID из .env (SPREADSHEET_ID)
- Чтение листа по названию из .env (SHEET_NAME)
- Эндпоинт: GET /api/data
- Формат ответа: JSON-массив объектов, где ключи - первая строка (заголовки), значения - остальные строки
- Обработка ошибок: если таблица недоступна - 500 с описанием, логирование в консоль
- Логирование всех запросов и ошибок (import logging)

## ДОПОЛНИТЕЛЬНЫЕ ТРЕБОВАНИЯ:
- Весь код с комментариями на английском или русском
- В коде использовать typing (type hints)
- Необходимые библиотеки храни в .venv (virtual environments)
- Разделить на функции, не мешать всё в кучу
- В config.py проверять, что все переменные окружения заданы
- В sheets_client.py сделать класс GoogleSheetsClient с методами get_data()
- Напиши инструкцию по деплою на Google Cloud Run (кратко, ссылки на документацию)
